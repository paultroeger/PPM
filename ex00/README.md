## Task 2
### What columns would you remove?
I would remove Name and SSN because they uniquely identify a person instantly.
### Why these columns and not the others?
I wouldn't remove the others because taken on their own they don't uniquely identify a person.
For example many people can be born on the same day and without other knowledge it would be impossible to identify one person from that information.
